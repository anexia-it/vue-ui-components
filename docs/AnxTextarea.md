The *anx-textarea* is very similar to the [*anx-input*](#anx-input). See  [*anx-input*](#anx-input) for more examples. Below is just a simple usage of the *anx-textarea*.

```html
<anx-textarea
    id="my-textarea"
    name="textarea"
    label="Textarea"
    assistive-text="This is a simple textarea"
    rows="5"
    rules="required"
/>
```
